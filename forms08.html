Cuadro de seleccion multiple

<form action="http://www.example.com/example.php">
    <p>Do you play any of the following instruments?
(You can select more than one option by holding
down control on a PC or command key on a Mac
while selecting different options.)</p>
    <select name="instruments" size="3" multiple="multiple">
    <option value="guitar" selected="selected">Guitar</option>
    <option value="drums">Drums</option>
    <option value="keyboard">Keyboard</option>
    <option value="bass">Bass</option>
    </select>
    </form>

Resultado:

Do you play any of the following instruments?
(You can select more than one option by holding
down control on a PC or command key on a Mac
while selecting different options.)

<select>: Muestra un cuadro de lista desplegable, permite seleccionar una opcion

  • <option>: Despliega un elemento que puede ser seleccionado
  • value: Indica el valor que es enviado al servidor
  • name: Indica el nombre del atributo
  • selected: Indica cual es el valor a ser seleccionado cuando la pagina es cargada
  • size: Indica el total de valores que seran mostrados de la lista
  • multiple: Indica si el usuario puede seleccionar mas de una opcion
